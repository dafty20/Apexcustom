# APEXカスタム ポイント集計ツール

## 概要
APEXのカスタムマッチ用ポイント集計ツールです。  
- 複数試合管理  
- 暫定/最終ランキング  
- 過去試合確認  
- 画像出力（16:9）  
- localStorage でデータ保持  

## 使い方
1. `index.html` をブラウザで開く  
2. ルール設定（キルポイント・順位ポイント）  
3. チーム追加・メンバー入力  
4. 「次の試合を作成」で試合結果を保存  
5. 「前の試合を確認」で直前試合確認  
6. 「カスタム終了」で最終ランキングを確定  
7. 「画像出力」で最終結果をPNG保存  

※ 背景画像を変更する場合はお好きな画像を `Apexrankingback.png` という名前に変更後、同フォルダに置いてください。
